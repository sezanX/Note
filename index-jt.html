import numpy as np

# Example system: Ax = b
A = np.array([[3, 1, 1],
              [1, 2, 2],
              [2, 1, 3]], dtype=float)
b = np.array([4, 3, 4], dtype=float)

n = len(A)
L = np.zeros((n, n))
U = np.zeros((n, n))

# LU Decomposition (Doolittle method)
for i in range(n):
    # Upper Triangular
    for k in range(i, n):
        U[i, k] = A[i, k] - sum(L[i, j] * U[j, k] for j in range(i))
    # Lower Triangular
    L[i, i] = 1
    for k in range(i+1, n):
        L[k, i] = (A[k, i] - sum(L[k, j] * U[j, i] for j in range(i))) / U[i, i]

print("L:\n", L)
print("U:\n", U)

# Solve Ly = b (forward substitution)
y = np.zeros(n)
for i in range(n):
    y[i] = b[i] - sum(L[i, j] * y[j] for j in range(i))

# Solve Ux = y (backward substitution)
x = np.zeros(n)
for i in range(n-1, -1, -1):
    x[i] = (y[i] - sum(U[i, j] * x[j] for j in range(i+1, n))) / U[i, i]

print("Solution x:", x)












Numerical Methods Explained

1. Bisection Method

Definition

The bisection method is a root-finding algorithm that repeatedly bisects an interval and selects a subinterval in which a root must lie, based on the Intermediate Value Theorem. It requires that the function changes sign over the interval [a, b].

Mathematical Example

Find a root of f(x) = x³ - x - 2 in [1, 2].
Since f(1) = -2 < 0 and f(2) = 4 > 0, a root exists in this interval.
The method computes midpoints: c = (a + b)/2, and updates the interval based on the sign of f(c).

Python Code

Bisection Method 
import math

def func(x):
    return x**3 - x - 2

def bisection(f, a, b, tol=1e-10, max_iter=100):
    if f(a) * f(b) >= 0:
        print("Bisection method fails: f(a) and f(b) must have opposite signs.")
        return None
    
    for i in range(max_iter):
        mid = (a + b) / 2
        f_mid = f(mid)
        
        if abs(f_mid) < tol:
            return mid
            
        if f(a) * f_mid < 0:
            b = mid
        else:
            a = mid
    
    print("Bisection method did not converge.")
    return None

root = bisection(func, 1, 2)
print("Root found:", root)

Code Explanation

  • func(x): Defines the target function f(x) = x³ - x - 2.
  • Initial check: Ensures f(a) and f(b) have opposite signs.
  • Loop: Computes midpoint mid and evaluates f(mid).
  • Update interval: Replaces a or b based on sign of f(mid).
  • Convergence: Stops when |f(mid)| < tol (here, 1e-10).

2. Newton-Raphson Method

Definition

An iterative root-finding method that uses the first derivative to approximate the root. Starting from an initial guess x₀, it uses the tangent line to converge quadratically under good conditions.

Mathematical Example

Solve f(x) = x² - 2 = 0 → root is √2 ≈ 1.414.
Derivative: f'(x) = 2x.
Iteration: x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} = x_n - \frac{x_n² - 2}{2x_n}.

Python Code

Newton-Raphson Method 
def f(x):
    return x**2 - 2

def df(x):
    return 2 * x

def newton_raphson(f, df, x0, tol=1e-6, max_iter=100):
    x = x0
    for _ in range(max_iter):
        fx = f(x)
        dfx = df(x)
        if dfx == 0:
            raise ValueError("Derivative is zero.")
        x_new = x - fx / dfx
        if abs(x_new - x) < tol:
            return x_new
        x = x_new
    raise RuntimeError("Did not converge")

root = newton_raphson(f, df, 1.0)
print("Root:", root)

Code Explanation

  • f(x) and df(x): Function and its derivative.
  • Iteration: Updates estimate using tangent line: x_new = x - f(x)/f'(x).
  • Stopping criterion: When successive estimates differ by less than tol.
  • Error handling: Raises error if derivative is zero (horizontal tangent).
  • Convergence: Quadratic near simple roots, but requires good initial guess.

3. Secant Method

Definition

A root-finding algorithm that uses a succession of roots of secant lines to approximate a root. It is similar to Newton-Raphson but does **not require the derivative**—instead, it uses a numerical approximation of the derivative from two previous points.

Mathematical Example

For f(x) = x³ - x - 2, choose initial guesses x₀ = 1, x₁ = 2.
Iteration formula:

x_{n+1} = x_n - f(x_n) \cdot \frac{x_n - x_{n-1}}{f(x_n) - f(x_{n-1})}

Python Code

Secant Method 
def f(x):
    return x**3 - x - 2

# Find interval where sign changes
x0, x1 = 0, 0
for i in range(10):
    if f(i) * f(i+1) < 0:
        x0, x1 = i, i+1
        break

# Secant iteration
while True:
    if f(x1) == f(x0):
        print("Division by zero")
        break
    x2 = x1 - f(x1) * (x1 - x0) / (f(x1) - f(x0))
    if abs(x2 - x1) < 1e-6:
        print(f"Root ≈ {x2:.6f}")
        break
    x0, x1 = x1, x2
Show more

Code Explanation

  • Initial guess: Automatically finds [x0, x1] where f changes sign.
  • No derivative needed: Uses finite difference (f(x1)-f(x0))/(x1-x0) as slope.
  • Update rule: Computes new estimate x2 using secant line through (x0,f(x0)) and (x1,f(x1)).
  • Convergence check: Stops when |x2 - x1| < 1e-6.
  • Robustness: Simpler than Newton-Raphson when derivative is unavailable.

4. LU Decomposition (Doolittle)

Definition

LU decomposition factors a square matrix A into a lower triangular matrix L and an upper triangular matrix U such that A = LU. It is used to solve systems of linear equations Ax = b efficiently.

Mathematical Example

Solve:

\begin{bmatrix} 3 & 1 & 1 \\ 1 & 2 & 2 \\ 2 & 1 & 3 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} 4 \\ 3 \\ 4 \end{bmatrix}
Step 1: Decompose A = LU.
Step 2: Solve Ly = b (forward substitution).
Step 3: Solve Ux = y (backward substitution).

Python Code

LU Decomposition (Doolittle) 
import numpy as np

A = np.array([[3, 1, 1],
              [1, 2, 2],
              [2, 1, 3]], dtype=float)
b = np.array([4, 3, 4], dtype=float)

n = len(A)
L = np.zeros((n, n))
U = np.zeros((n, n))

# Doolittle: L has 1s on diagonal
for i in range(n):
    # Compute U[i, k]
    for k in range(i, n):
        U[i, k] = A[i, k] - sum(L[i, j] * U[j, k] for j in range(i))
    # Compute L[k, i]
    L[i, i] = 1.0
    for k in range(i+1, n):
        L[k, i] = (A[k, i] - sum(L[k, j] * U[j, i] for j in range(i))) / U[i, i]

# Forward substitution: Ly = b
y = np.zeros(n)
for i in range(n):
    y[i] = b[i] - sum(L[i, j] * y[j] for j in range(i))

# Backward substitution: Ux = y
x = np.zeros(n)
for i in range(n-1, -1, -1):
    x[i] = (y[i] - sum(U[i, j] * x[j] for j in range(i+1, n))) / U[i, i]

print("Solution x:", x)
Show more

Code Explanation

  • Matrix decomposition: Constructs L (unit lower triangular) and U (upper triangular) using Doolittle’s method.
  • Forward substitution: Solves Ly = b for y (since L is lower triangular).
  • Backward substitution: Solves Ux = y for x (since U is upper triangular).
  • Efficiency: Once A = LU is computed, multiple right-hand sides b can be solved quickly.
  • Assumption: Matrix A must be square and non-singular (and ideally diagonally dominant for stability).
